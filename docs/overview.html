Machine Learning models for Cystic Fibrosis - Tristan Trébaol

Description

This is the documentation of the my master's thesis code:

  • - Thesis title: Bayesian inference with Expectation Maximisation for the characterisation of antibiotic treatment recovery in Cystic Fibrosis
  • - Side project title: Estimation of the variability in Cystic Fibrosis patients FEV1 lung function measurements

    • Hosted at the University of Cambridge, supervised at EPFL

    Use

  • View report and download the source code's latest version on GitHub
  • Requires Matlab 2021a.

    • Content

    The code is divided into 5 parts, arranged by directory:
    • exploration Project Breathe data exploration.
    • recovery Main project.
    • recovery/updatedModel Main project (updates on the initial SMARTCARE model).
    • FEV1variability Side project.
    • helperFunctions Functions common to at least two other matlab scripts.
    • smartcare code from the SMARTCARE study.
